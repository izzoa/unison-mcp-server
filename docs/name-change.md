# Unison MCP Name Change

Unison MCP was previously called PAL MCP (Provider Abstraction Layer), and before that, Zen MCP. We renamed to Unison - Providers Together - to better reflect the collaborative, multi-provider nature of the project. The software and workflows are the same.

Due to the change of name, you may need to run `run-server.sh` again to setup the new connection, and re-visit any `PAL` name used within `.env` and change it to `UNISON`.
